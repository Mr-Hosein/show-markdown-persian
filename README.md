# 📖 دفترچه — نمایش‌گر مارک‌داون فارسی

یه صفحه‌ی HTML تک‌فایلی و کاملاً آفلاین برای خوندن راحت فایل‌های مارک‌داون فارسی، بدون به‌هم‌ریختگی جهت متن (RTL) و بدون نیاز به هیچ اتصال اینترنتی یا کتابخونه‌ی بیرونی.

## چرا این پروژه؟

نمایش فایل‌های `.md` فارسی تو خیلی از ادیتورها و پیش‌نمایش‌های معمولی به‌هم می‌ریزه، مخصوصاً وقتی متن فارسی با کد، لینک، یا اعداد قاطی می‌شه. این پروژه یه محیط خواندنی مخصوص متن‌های فارسی می‌سازه:

- جهت متن همیشه درست RTL هست
- بلوک‌های کد و کدهای درون‌خطی به‌صورت جدا LTR نمایش داده می‌شن (بدون به‌هم‌ریختن ترتیب کاراکترها)
- فهرست مطالب خودکار از روی عنوان‌ها ساخته می‌شه
- نوار پیشرفت مطالعه بالای صفحه

## ویژگی‌ها

- **بدون وابستگی بیرونی** — هیچ CDN یا کتابخونه‌ای لود نمی‌شه؛ پارسر مارک‌داون داخل خود فایل نوشته شده
- **کاملاً آفلاین** — فقط با دوبار کلیک روی فایل HTML باز می‌شه، حتی بدون اینترنت
- **پشتیبانی از مارک‌داون رایج**: عنوان‌ها (`#` تا `######`)، لیست‌های نقطه‌ای و شماره‌دار (با یک سطح تودرتو)، لینک و تصویر، نقل‌قول، جدول، بلوک کد، بولد/ایتالیک/خط‌خورده، کد درون‌خطی
- **باز کردن فایل دیگر** — دکمه‌ای برای انتخاب و نمایش هر فایل `.md` دیگه از روی کامپیوتر، مستقیم تو همون صفحه
- **واکنش‌گرا** — رو موبایل، فهرست مطالب به یه دکمه‌ی شناور تبدیل می‌شه

## استفاده

فقط فایل `persian-python-reader.html` رو دانلود کن و تو هر مرورگری بازش کن. همین.

```
git clone <آدرس ریپازیتوری>
cd <نام-پوشه>
# فایل HTML رو با دوبار کلیک یا از طریق مرورگر باز کن
```

می‌خوای آنلاین هم در دسترس باشه؟ می‌تونی از **GitHub Pages** استفاده کنی:
1. تو تنظیمات ریپازیتوری، بخش Pages رو باز کن
2. شاخه (branch) و فولدری که فایل توشه رو انتخاب کن
3. لینک صفحه رو بگیر و به اشتراک بذار

## محدودیت‌ها

پارسر مارک‌داون این پروژه یه پیاده‌سازی سبک و اختصاصیه، نه یه پیاده‌سازی کامل CommonMark. مواردی مثل جدول‌های تودرتو، پانویس (footnote)، یا HTML خام داخل مارک‌داون پشتیبانی نمی‌شن.

## ساختار فایل

همه‌چیز — استایل، منطق نمایش، و حتی خودِ متن مارک‌داون (به‌صورت base64) — داخل یه فایل HTML واحده. برای همینه که این‌قدر قابل‌حمل و ساده برای اشتراک‌گذاریه.

## English (short)

A single, dependency-free HTML file for reading Persian (Farsi) markdown documents with correct RTL text flow, isolated LTR code blocks, an auto-generated table of contents, and a built-in file picker to open any other `.md` file locally. No CDN, no network access required — just open the file in a browser.
